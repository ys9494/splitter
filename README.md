## 🧮 프로젝트 소개

🧾 **모임 후 복잡한 결제내역을 쉽게 정산하는 기능을 제공하는 서비스**

-   여러 개의 결제 내역을 한 번에 정산할 때 이용하기 좋은 서비스
-   ex) N차 모임, 여행 정산 등
-   모임 별로 인원 구성이 다른 N차 모임을 했을 때 이용하기 편리한 정산 서비스이다.
-   모임에서 일부 인원만 따로 주문한 항목(ex) 술, 고가의 아이템 등)이 있을 때 해당 항목을 추가하여 한 번에 정산할 수 있다.
